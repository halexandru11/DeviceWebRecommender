How to navigate though the app

1. Set Preferences

Set your preferences by hovering mouse over Menu button and selecting Profile. Here you can select the devices you want to be updated about and select the right choices acording to your liking.

2. Set Application's color theme

Go to Menu ⇨ Themes and set desired color theme from the schemas given.

3. Navigate through the app

Go to Menu -> Home. If you’re signed in, you’ll be given recommendations of products based on the preferences you have set in you Profile page. If you are not authenticated, you’ll get a selection of products based on trending/ general preferences of the users of the app. You can choose the sites from where you can receive recommendations by deselecting the sites you’re not interested in. By clicking on the MENUBUTTON button you can add different sites from the list from where you’ll get recommendations as well.

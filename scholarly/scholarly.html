DeWR (Device Web Recommender) - Gimme App

DeWR is a web application that recommends electronic devices based on user preferences or on general requirements. It is a part of the Gimme App project.

Use-case diagram of user Gimme App

The Gimme app is designed to work as a browser extension, that suggests to the user electronic devices, based either on the user preferences (in case the user is present in the user data base and logged in the app), or on general facts if the user is not authenticated.

The purpose of the app is to facilitate the selection of options that a user faces when seeking to purchase a new electronic device, scavenging for the products most popular at the moment, from trusted websites, that the user can select to get recommendations from.

Starting the app, the page that is first opened is the product recommendations page. Here you see a collection of electronic devices(E.G. laptops, smartphones, tablets, keyboards, etc...). Pressing on one product leads you to another page, where you can see more photos of the product, the price, a small description and the link to the merchant page.

The user can also access the Menu button on top right, where he can change the theme of the app, the language and the currency. The user can also access the About page, where he can see the information about the app and the developers. On this page there are also testimonials from users of the app.

In case of any questions about the way that the app works, the user can consult the help page, that gives explainations reagarding the usage of the app.
